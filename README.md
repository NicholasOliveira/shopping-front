# Shopping Cart APP

#### Demo: [LINK TO DEMO](https://musing-stonebraker-a4df74.netlify.app/)

## Scripts Available

for run project use command down:
### `npm run start`

open main components view easy
### `npm run storybook`

run all tests and print coverage
### `npm run test`

create build project to public.
### `npm run build`
